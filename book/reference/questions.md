# FAQs

:::{dropdown} Q: Do I need to know Python to participate? 

**A:** Since the tutorials will be in Python, we recommend participants have some familiarity with Python syntax so that they are able to follow the content. We recommend reviewing this introductory [Python programming lesson](https://swcarpentry.github.io/python-novice-gapminder/) before the workshop. You are welcome to use any programming language for project work.
:::

:::{dropdown} Q: I want to learn to use PyTorch to analyze my data. Is this hackweek right for me?

**A:**  No, this event focuses on Large Language Models (LLMs) and their application in
a typical research workflow. We will be using gen-AI to reduce friction in 
building software and accessing data. We encourage you to contact the eScience 
Institute for other tutorials that cover AI/ML applications for research data
analysis.
:::

:::{dropdown} Q: I have never used LLMs for research. Is this hackweek right for me? 

**A:** This workshop is designed for participants who have some experience using LLMs for some component of scientific work. This can include chat-based inquiry and code development or more fully-orchestrated agentic workflows. If you have never used LLMs in this capacity, we encourage you to wait for a future hackweek where we will be offering tutorials that cover LLM fundamentals. 
:::

:::{dropdown} Q: I currently do not have an account for an LLM service. Will this be provided? 

**A:** Yes, we will be provisioning a variety of LLM applications for you to use throughout the week. These will be offered free of charge. 
:::

:::{dropdown} Q: Will you be using open LLMs or those that are commercially provided?  

**A:** Our provisioned LLMs will include both open and commercial models. Our goal is to position you to build toolkits that are agnostic to the LLM application. Along the way we will discuss the pros and cons of open versus closed models.  
:::

:::{dropdown} Q: It's important to me that AI gets used carefully for scientific research. Will you be addressing this topic? 

**A:** Yes; this is a core motivation for our hackweek. We are coming together to explore how responsible gen-AI can support scientific progress by focusing on topics of reproducibility, robustness and security. We hope to work together with you in shaping how scientists can explore potential uses of these tools going forward and build a network of responsible gen-AI users and leaders.     
:::

:::{dropdown} Q: Am I required to participate in all tutorials and activities?

**A:** Yes, participation is mandatory for all days that you have signed up for. We expect all participants to be actively engaged in all workshop activities. 
:::
